# База клинических и генетических данных по болезни Паркинсона

Веб-приложение для работы с тремя связанными таблицами:
- metadata — пациенты с БП и их характеристики
- variants — аннотация генетических вариантов (VEP)
- genotypes — связь пациент–вариант

## Структура

- `database.py` — подключение к SQLite и инициализация таблиц
- `models/` — структура таблиц
- `schemas/` — валидация форм и CSV
- `crud.py` — операции с базой данных
- `app.py` — Flask маршруты
- `templates/` — HTML-страницы

## Запуск

```bash
pip install -r requirements.txt
python app.py
```
